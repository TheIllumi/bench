# ADR 0003 --- Projects Are First-Class Citizens

**Status:** Accepted

## Context

Tasks, notes, and resources require context.

## Decision

Projects are the primary organizational unit in Bench.

Every meaningful piece of work belongs to exactly one Project.

## Consequences

-   Tasks cannot exist without a Project.
-   Notes cannot exist without a Project.
-   Resources exist only within Projects.
-   Projects become the center of the user workflow.
